# leonim/im-server 


---

## 简介
`leonim/im-server` 基于 **Workerman + GatewayWorker** 封装的即时通讯服务。 


---

## 安装

1. 首先确保你已经安装 webman-admin：

   ```bash
   composer require -W webman/admin  
   ```
   
2. 然后安装 leonim/im-server：

   ```bash
   composer require leonim/im-server
    ```
   
3. webman 中执行启动命令：
    ```bash
    php start.php start
    ```
